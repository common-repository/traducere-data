Acest plugin va traduce data in limba romana.

== Description ==

Acest plugin va traduce data in limba romana, atat pentru articole cat si pentru cometarii. De asemenea va traduce in limba romana si numele lunilor afisate la arhiva

== Installation ==

1. Downloadati arhiva si dezarhivati-o
2. uploadati folderul in directorul cu plugin-uri (de obicei wp-content/plugins/)
3. Activati pluginul in panoul de control
